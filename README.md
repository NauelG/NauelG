## 💡 How I think about this craft

The most interesting work happens where **technology meets business**. Technology should:

- solve real problems
- improve how organizations operate
- unlock measurable opportunities
- create durable value

I'm allergic to hype and to its mirror image — reflex skepticism. AI doesn't add clarity; it demands it. Tools age fast; judgment doesn't.

> **Technology is a means, not an end.**
